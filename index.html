What is ONA

The org chart is a map of authority. It says nothing about how work actually gets done.

In every organisation, the real work flows through an invisible web of conversations, trust and collaboration. Organisational Network Analysis makes that web visible — turning relationships into data you can see, measure and act on.

Where the hierarchy ends, the network begins. That's where we look.
